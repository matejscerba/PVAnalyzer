Detailed Description

Represents frame-wise parameter, e.g. position of left ankle throughout whole processed video.

Holds this parameter's value for each frame.

Write parameter's value for given frame number.

If value is valid, write it to output stream followed by unit if it is supposed to be written. Write nothing if unit is not supposed to be written and value is invalid. Write question mark followed by unit if unit is supposed to be written and value is invalid.
